Voici toutes les modifications que vous avons realisé pour la version 4 de SimpleChat :

- Graphic User Interface Client :
	- Un JTextPane pour afficher les messages du serveur
	- Un JTextField pour ecrire un message au serveur
	- Un JButton pour envoyer le message du JTextField au serveur
	- Une JMenuBar pour se connecter/deconnecter/changer de pseudo/port/hôte
	- Un JTextPane pour afficher la liste des personnes connectées
	- Un JRadioButton pour afficher (ou non) l'heure du message
	- Deux JScrollPane pour afficher une scrollbar dans chaque JTextPane
	
	Le JTextField et le JButton ont été placé dans un JPanel utilisant un BoxLayout utilisant.
	Tous les elements ont ensuite été placé sur la JFrame grâce aux positions du BorderLayout (CENTER/EAST/SOUTH).
	
- Graphic User Interface Server :
	- Des JButton pour envoyer les commandes start/stop/close/setport/quit au serveur 
	- Un JLabel pour afficher le port du serveur
	- Un JTextPane pour afficher les messages du serveur
	- Un JTextField pour ecrire un message aux clients ou au serveur
	- Un JButton pour envoyer le message du JTextField aux clients ou au serveur
	- Une ScrollPane pour afficher une scrollbar dans le JTextPane
	
	Les JButton et le JLabel ont été placé dans deux JPanel utilisant un GridLayout(3,1) afin d'aligner correctement ces elements.
	Le JTextField et le JButton ont été placé dans un JPanel utilisant un BoxLayout.
	Tous les elements ont ensuite été placé sur la JFrame grâce aux positions du BorderLayout (NORTH/CENTER/SOUTH)
	
- UserList :
	Nous avons rajouté la liste des utilisateurs connectes.
	Pour realiser cela, il a fallu modifier la methode run() de AbstractServer.
	Lors du lancement de run(), on initialise la variable nbClients à 0.
	Dans la boucle while, on verifie si le nombre de clients a change.
	Si c'est le cas, alors on envoit à tous les clients la liste des clients connectes, prefixes du symbole $.
	De plus, la methode display() de ClientGUI teste si le message passe en parametre debute par le symbole $
	Si c'est le cas, alors le JTextPane affichant la liste des clients est mis a jour.
